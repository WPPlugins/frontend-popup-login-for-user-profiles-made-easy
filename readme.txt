Frontend Popup Login for User Profiles Made Easy.

== Description ==
[Frontend Popup Login for User Profiles Made Easy](https://goo.gl/SV6oDy)  is a free addon provided by UPME for adding popup based login forms to WordPress sites. Now users donâ€™t need to access separate page for login. You can place a login link on site header, sidebar or within a post and let users login through ajax without page refresh.

Please note that this is an addon for  [User Profiles Made Easy](https://goo.gl/w716O3) plugin and you need at least UPME version 2.2.03 to use it.

> **Main Features**
>
> * Front-end login form.
> * AJAX based login support.
> * Login using a popup window.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/upme-frontend-popup-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the docs at https://goo.gl/SV6oDy to use frontend popup login.
